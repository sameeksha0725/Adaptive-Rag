# Code of Conduct

This project follows the Contributor Covenant Code of Conduct. By participating, you agree to abide by the following guidelines:

- Be respectful and professional.
- Use constructive language in issues and PRs.
- No harassment, hate speech, or abusive behavior.

If you experience or witness violations, please contact the maintainers.

Full policy: https://www.contributor-covenant.org/version/2/0/code_of_conduct/
